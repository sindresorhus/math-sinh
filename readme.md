# math-sinh

> ES2015 [`Math.sinh()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/sinh) [ponyfill](https://ponyfill.com)


## Install

```
$ npm install --save math-sinh
```


## Usage

```js
var mathSinh = require('math-sinh');

mathSinh(1);
//=> 1.1752011936438014
```
